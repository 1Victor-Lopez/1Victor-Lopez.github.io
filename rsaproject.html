The purpose of this program is to Encrypt and Decrypt whatever message you put into it using a value of e, n, and d. In the video, it shows how the program encrypts using the e and n values, and how it decrypts using the n and d values. The user inputs the message they want to encrypt but has to input an e and n value. When finished and clicked on the “Encrypt” button, it automatically transfers over to decrypt section. In this section the user puts in an n and d value then clicks on “Decrypt” button which then decrpyts the encrypted message.

The “Encbutton()” function calls the “encrypt(e, n)” function. This button encrypts the message put into the textbox. The “encrypt(e, n)” function encrypts any message put into the the text box “textE”. When the “Encrypt” button is clicked, it calls the “Encbutton()” function which uses the entries, “entryN” and “entryE”, to use as the n and e so that the code can encrypt. Then it calls the “encrypt(e, n)” function which encrypts the message.
